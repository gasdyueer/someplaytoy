成绩规则

总成绩

总成绩(100分)= 平时成绩(40分)+ 平时测试成绩(10分)+ 见面课成绩(10分)+ 期末考试成绩(40分)

  • 1、总成绩在考试截止时间48小时之后统一发布；
  • 2、学校规定，总成绩不合格将不允许补考；
  • 2、若期末考试有主观题，则由老师批阅后手动发布成绩

平时成绩(40分)

平时成绩=学习进度成绩（15分）+ 学习习惯成绩（5分）+ 我的互动成绩（5分）+知识点掌握度（ 15分)

学习结束之前观看章节目录中的视频可获得进度成绩；
合理规划学习行为、共享课多进行有效的问答讨论可获得习惯成绩与互动成绩；
通过知识下的“提升掌握度”入口作答题目即可获得掌握度，学习周期内可无限次重复作答，以最新一次结果为准。

见面课成绩(10分)

见面课成绩为学生所有签到成绩、表现成绩与随堂测验成绩的总和

  • 1、签到成绩-学生通过参加见面课时完成对应的签到后获得的成绩
  • 2、表现成绩-老师会根据学生参与见面课的表现给予相应的分数成绩
  • 3、测验成绩-见面课后，老师会布置随堂测验，完成测验获得相应的成绩。

平时测试成绩(10分)

平时测试成绩=所有平时测试得分之和/所有平时测试总分*10

期末考试成绩(40分)

期末考试成绩=[期末考试分/期末考试总分]*100*40%
